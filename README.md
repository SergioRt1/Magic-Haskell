# Magic-Haskell
 Project in haskell
#### By Sergio Andrés Rodríguez Torres

## Requirements

stack 1.9

cabal 1.24.2

## buld

`git clone https://github.com/SergioRt1/Magic-Haskell.git`

`cd Magic`

`cd CLI`

`cabal install random`

`cabal install random-shuffle`

`cabal install fclabels`

`ghc --make *.hs`

## Run

`./Main` en linux

`Main.exe` en Windows
